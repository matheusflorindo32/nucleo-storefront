# Núcleo TADS Store

## Descrição
Projeto desenvolvido para a **Semana 12 / Etapa 1** do Projeto Integrador TADS Store, com foco em **componentização em React**.

## Como rodar
```bash
npm install
npm run dev
```

## Funcionalidades implementadas
- Projeto criado com React + Vite
- Componentes reutilizáveis
- Layout com `props.children`
- Cabeçalho com identidade visual
- Vitrine de produtos
- Cards reutilizáveis
- Props
- Composição
- Array fixo de produtos
- Renderização com `.map()`
- Uso de `key`
- Renderização condicional do selo de frete grátis
- CSS próprio com identidade visual clara
